Create a program which takes the sum of the multiples of two given numbers until a certain point. 

If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.
Find the sum of all the multiples of 3 or 5 below 1000. Complete the given function so that it meets the given specifications.
